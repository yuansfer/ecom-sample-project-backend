# Ecom Sample Project (Backend)


## Install dependencies
```bash
yarn install
```
## Run
```bash
yarn nodemon
```
## Create Database
```bash
Name with "yuansfer"
```
## Run Migration to Create Database Schemas
```bash
yarn sequelize db:migrate
```
## Run Seeders To Create Default Customer and Merchant User
```bash
yarn sequelize db:seed:all
```
