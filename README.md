# Ecom Sample Project (Backend)


## Installation

```bash
yarn install
```

```bash
Create Database Name with "yuansfer"
```

```bash
yarn sequelize db:migrate  (To Create all schema)
```

```bash
yarn sequelize db:seed:all  (To Create default merchant and customer user)
```
